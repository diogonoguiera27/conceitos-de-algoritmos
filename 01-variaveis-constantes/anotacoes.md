# Variáveis, Constantes e Tipos de Dados

## O que são variáveis?

Variáveis são espaços na memória usados para guardar valores que podem mudar durante a execução do programa.

## O que são constantes?

Constantes são valores que não devem mudar durante a execução do programa.

## O que são tipos de dados?

Tipos de dados definem qual tipo de informação uma variável ou constante pode armazenar.

Exemplos:

- string: textos
- number: números
- boolean: verdadeiro ou falso
- null: ausência intencional de valor
- undefined: valor ainda não definido
