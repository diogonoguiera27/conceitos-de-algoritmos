# Tipos de Dados

Nesta pasta vou estudar os principais tipos de dados usados em algoritmos e programação.

## O que são tipos de dados?

Tipos de dados definem qual tipo de valor uma variável ou constante pode armazenar.

Exemplos:

- Texto
- Número
- Verdadeiro ou falso
- Valor vazio
- Valor não definido

## Objetivo desta aula

Entender como identificar e usar os principais tipos de dados em exemplos práticos.
